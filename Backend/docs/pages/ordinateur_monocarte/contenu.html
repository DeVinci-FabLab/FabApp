    Présentation et comparatif des modèles¶

    Bien choisir sa carte

    • Pour faire tourner des modèles demandant du gpu, il faut aller voir les cartes Nvidia

    Raspberry pi like¶

    Avantages

    • 8 à 16 Go de RAM DDR4

    • 2.4 GHz

    • Open source

    • wifi et bluetooth

    Défaults

    • 5V/4A

    • 100$ - 150 $

    Orange Pi 5B

    Avantages

    • Open-source

    • 30$

    Défaults

    • 4go de RAM max

    Tritium

    Potato

    Défaults

    • 90 € (quand disponible)

    • 8go de RAM max

    • Rupture de stock

    • Prix élevé

    • Non libre

    Rasperry PI

    Carte avec GPU¶

    Attention Open Source

    Sur les cartes Jetson, Nvidia est propriétaire

    Avantages

    • 8go de RAM

    • 1024 CUDA cores

    • 32 tensor cores

    Défaults

    • 400 €

    • pas de cartes wifi

    Jetson Orin Nano

    Avantages

    • 128-core NVIDIA

    Défaults

    • 4go de RAM

    • 150€

    • Pas de carte réseau

    Jetson Nano

    Avantages

    • 2048 CUDA cores

    • 64 Tensor cores

    • 32GB 256-bit LPDDR5

    Défaults

    • 1 385€

    Jetson Nano

    Compute Module¶

    Puissance gpu Nvidia

    Nous pensons qu’il est trop compliqué de prendre un compute module qui rajoute de la complexité aux ordinateurs monocartes

    Avantages

    • La plus utilisée

    • 8go de RAM

    • Compatibilité avec pleins de boards

    Défaults

    • raspberry pi en plus volumineux avec la board

    • Complexité

    Avantages

    • 8go de RAM DDR4

    • ARM + Risc V fully open source (not for arm)

    Défaults

    • pas d’upgrde possible
